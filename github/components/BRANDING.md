# Identidad de Marca (RQT2)

Este documento define las reglas para los isotipos de **RQT2**. 

## Concepto

El logotipo de [RQT2](https://github.com/RQT2/rqt2-components/blob/main/assets/branding/logo.xcf) se basa en la modificación de una `T` usando la fuente **Noto Sans Bold**: Una T horizontal se extendio y curvó para dibujar 2/3 de `Q`. Al final se agregó un `2` en formato de display para que en conjunto con la T modificada parezca una `R`.

### Variantes
| Variante | Archivo | Enfoque |
| :--- | :--- | :--- |
| **Full Color** | `logo-main-color.svg` | Uso general y splash screens. |
| **Light** | `logo-main-dark.svg` | Para fondos oscuros (ej. modo oscuro de la IDE). |
| **Dark** | `logo-main-light.svg` | Para fondos claros y documentos impresos. |

## 2. Icono de Aplicación (App Icon)

Para que RQT2 se integre de forma nativa, el icono debe seguir reglas específicas de márgenes y forma.

### Área Segura y Márgenes
Basado en los estándares de **[GNOME AdwaitaDa](https://developer.gnome.org/hig/guidelines/app-icons.html)**:
- **Canvas Base:** [template.svg](https://github.com/RQT2/rqt2-components/blob/main/assets/branding/template.svg).



### Estilo Visual
- **Fondo:** Con forma "Squircle" (cuadrado redondeado).
- **Radio de curvatura:** 18.75%.
